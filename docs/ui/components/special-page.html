# special-page

异常页面，例如异常反馈页和结果页。

# 1. DOM结构

  • q-page：结果页
  • q-page__wrap：页面内容
  • q-page__wrap__img：页面图标
  • q-page__wrap__text：非结果页的标题
  • q-page__wrap__title：结果页的主标题
  • q-page__wrap__sub-title：结果页的副标题
  • q-page__bottom：页面底部

# 2. 样式

  • q-page--result：结果页
