# radio

基础输入框组件，特殊的list item ，建议配合 list 容器使用

# 1. DOM 结构

  • pro-radio
    • pro-radio__control: 'radio'类型的input
    • pro-radio__pic: label前置元素
    • pro-radio__text: label提示
    • pro-radio__pick-icon: 勾选后的图标元素
